## 💻 Commit 규칙

```
{태그}: [문제_제목] {커밋 메시지}
```

| 태그     | 설명                                |
|:------------|:--------------------------------|
| feat     | 코드 구현                           |
| refactor | 코드 수정                           |
| comment  | 주석 추가(코드 변경 X) 혹은 오타 수정 |
| docs     | README와 같은 문서 수정             |
| merge    | merge                              |
| rename   | 파일, 폴더명 수정 혹은 이동          |

## 💻 PR 규칙
```
[날짜] {문제 제목} - {기타 내용}
```

- 💡 예시: `[231106] 두수의 차 - 문제풀이 아이디어는 ~~`
